# AGENTS

This project is the public landing site for Qatari Virtual, a Virtual Airline in Infinite Flight.

## Project quick facts
- Framework: Astro + Tailwind (Vite)
- Pages live in `src/pages/`
- Shared layout in `src/layouts/Layout.astro`
- Static assets in `public/`

## Working agreements
- Keep text and visuals consistent with the Qatari Virtual brand.
- Prefer reusing existing layout and style patterns.
- Use images from `public/images/` unless requested otherwise.
- Avoid introducing new dependencies without a clear need.

## Local development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Production build: `npm run build`
4. Preview build: `npm run preview`

## Notes for agents
- Update copy to reflect Qatari Virtual as a Virtual Airline in Infinite Flight.
- When adding new sections, ensure mobile layouts stay readable.
